# Sidely – Chrome 版 ChatGPT 侧边栏

Sidely 是你在 Chrome 中的专属 AI 助手。
告别频繁切换标签页或分心。这个轻量级的 ChatGPT 侧边栏扩展把 ChatGPT 直接带到浏览器侧边栏，一次点击即可开启。

🧩 功能概览
Sidely 将 OpenAI 的 ChatGPT 无缝集成进 Chrome。
无需离开当前页面，就能聊天、头脑风暴、概括文本、翻译或生成灵感。与其他工具不同，它直接连接到你的 ChatGPT 账号，让数据始终私密、安全，无需额外账号或密码。

⚙️ 主要特性
- 通过 Chrome 侧边栏，在任意标签页即时访问 ChatGPT
- 直接使用你已有的 ChatGPT 会话——无需注册或第三方登录
- 轻巧迅速：为速度和低资源占用而生
- 安全可靠：聊天数据和凭证永远不会离开你的浏览器
- 多场景助手：概括文章、解释代码、撰写邮件、翻译文本或激发创意
- 支持 GPT-5 和 GPT-4（取决于你的 OpenAI 订阅）

💡 为什么选择 Sidely？
无论是写邮件、写代码、做研究还是学习新知识，Sidely 都能帮你节省时间。无需再在新标签页打开 chat.openai.com，也不用来回复制粘贴。
它是一个专注且无干扰的效率工具，适合经常使用 ChatGPT 的人。
Sidely 让 ChatGPT 就像 Chrome 内置功能一样自然。

🔒 隐私优先的设计
你的 ChatGPT 会话通过内嵌界面直接运行在 Chrome 侧边栏中。
Sidely 不会收集、存储或传输你的对话、凭证或使用数据。
所有认证都只在你与 OpenAI 之间完成。
